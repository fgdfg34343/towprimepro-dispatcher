## 🚀 ЧТО СДЕЛАТЬ СЕЙЧАС:

### Шаг 1: Обновите правила Firestore

В Firebase Console → **Firestore Database** → **Rules**

Скопируйте и вставьте из файла `firestore.rules`:

```javascript
// Чаты поддержки
match /supportChats/{chatId} {
  allow read: if isAuthenticated();
  allow create: if isAuthenticated();
  allow update: if isAuthenticated();
  
  match /messages/{messageId} {
    allow read: if isAuthenticated();
    allow create: if isAuthenticated();
    allow update: if isAuthenticated();
  }
}
```

Нажмите **Publish**!

### Шаг 2: Перезапустите dev сервер

```bash
# Ctrl+C чтобы остановить
npm run dev
```

### Шаг 3: Проверьте диспетчерскую

1. Откройте диспетчерскую
2. Перейдите на вкладку **"Поддержка"** (иконка 💬)
3. Вы увидите:
   - **Левая колонка**: Список чатов (пока пусто)
   - **Центр**: "Выберите чат"
   - **Правая колонка**: (появится при выборе чата)

### Шаг 4: Создайте тестовый чат через Firebase Console

Для проверки создайте тестовый чат вручную:

1. Firebase Console → Firestore → **Start collection**
2. Collection ID: `supportChats`
3. Document ID: (Auto-ID)
4. Добавьте поля:

```
chatId: <auto-generated>
driverId: "XVhFITK6PWgzWa8l3BLftJQNGCx2"
driverName: "yk y"
driverPhone: "+7hhhhhhhhhh"
status: "active"
lastMessage: "Здравствуйте, нужна помощь"
lastMessageFrom: "driver"
lastMessageTime: <current timestamp>
unreadByDispatcher: 1
unreadByDriver: 0
createdAt: <current timestamp>
updatedAt: <current timestamp>
```

5. Нажмите **Save**

6. Создайте подколлекцию `messages`:
   - Откройте созданный документ
   - **Start collection**: `messages`
   - Document ID: (Auto-ID)
   - Поля:

```
chatId: <ID родительского документа>
senderId: "XVhFITK6PWgzWa8l3BLftJQNGCx2"
senderName: "yk y"
senderType: "driver"
text: "Здравствуйте, нужна помощь"
isRead: false
createdAt: <current timestamp>
```

7. **Обновите диспетчерскую** (Cmd+R)
8. ✅ Чат должен появиться в списке!
9. ✅ Нажмите на чат - откроется окно с сообщением
10. ✅ Попробуйте ответить - сообщение добавится в Firebase
