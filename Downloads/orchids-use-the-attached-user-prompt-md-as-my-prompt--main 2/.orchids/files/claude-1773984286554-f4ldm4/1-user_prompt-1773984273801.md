Нужно разработать современный продающий сайт для сервиса эвакуации автомобилей, в едином стиле с уже существующим мобильным приложением клиента.

Задача сайта:
Сайт должен не просто рассказывать о компании, а сразу конвертировать посетителя в заявку. Главный сценарий: клиент заходит на сайт, сразу видит красивый калькулятор эвакуации, вводит адрес “откуда забрать”, адрес “куда доставить”, выбирает условия перевозки, сразу видит итоговую цену и может оформить вызов эвакуатора.

Общие требования по стилю:
- Дизайн должен быть премиальный, современный, стильный и дорогой на вид.
- Визуальный стиль: светлый фон, акцентный оранжевый цвет, чистая типографика, аккуратные карточки, мягкие тени.
- На сайте обязательно использовать 3D-визуал или 3D-like элементы:
  - 3D эвакуатор
  - 3D машина
  - объёмные иллюстрации, мягкие световые акценты, глубина
  - анимации появления, hover-эффекты, плавные transitions
- Важно избежать шаблонного и дешёвого внешнего вида.
- Сайт должен выглядеть как современный high-conversion landing page + web app калькулятор.

Главный экран:
На первом экране пользователь должен сразу видеть:
- сильный заголовок
- короткое УТП
- калькулятор стоимости эвакуации
- 3D визуал эвакуатора/автомобиля
- CTA-кнопку “Вызвать эвакуатор”

Основной сценарий калькулятора:
1. Поле “Откуда забрать”
2. Поле “Куда доставить”
3. Выбор типа машины:
   - Легковая
   - Кроссовер
   - Внедорожник
   - Микроавтобус
   - Мотоцикл
   - Другое
4. Выбор количества заблокированных колёс:
   - 0
   - 1
   - 2
   - 3
   - 4
5. Выбор типа эвакуатора:
   - Ломаная платформа
   - Сдвижная платформа
   - Манипулятор
6. После выбора всех параметров пользователь сразу видит рассчитанную стоимость
7. После этого он нажимает кнопку “Вызвать эвакуатор” и оставляет контактные данные

Логика расчёта цены:
- Подача для обычных машин: 3500 ₽
- Подача для коммерческих машин: 5500 ₽
- 1 км для обычных машин: 110 ₽
- 1 км для коммерческих машин: 130 ₽
- 1 заблокированное колесо: 800 ₽
- Доплата за сдвижную платформу: +500 ₽
- Доплата за манипулятор: +1500 ₽
- Для расчёта расстояния использовать маршрут по дорогам, а не прямую линию
- Пользователь должен видеть итоговую сумму сразу в интерфейсе

Структура сайта:
1. Hero section с калькулятором и 3D-визуалом
2. Блок “Как это работает”
3. Блок преимуществ:
   - Круглосуточно 24/7
   - Быстрая подача
   - Прозрачная цена
   - Поддержка
4. Блок типов эвакуаторов
5. Блок часто задаваемых вопросов
6. Блок контактов
7. Футер

Функциональные требования:
- Адаптивность: desktop, tablet, mobile
- Отличная мобильная версия
- Быстрая загрузка
- Удобная форма заявки
- Валидация полей
- Интеграция карты и адресного автокомплита
- Расчёт маршрута по улицам
- Возможность подключения CRM / Telegram / WhatsApp / backend API
- После отправки заявки показывать красивое подтверждение

Технические ожидания:
- Использовать современный стек
- Чистая архитектура
- Компонентный подход
- Код должен быть пригоден для дальнейшего масштабирования
- Подготовить сайт так, чтобы потом можно было подключить реальные API, оплату и личный кабинет

Особое требование:
Сайт не должен быть похож на обычный шаблонный лендинг. Нужен сильный визуал, ощущение доверия, качества и современного сервиса. Интерфейс должен подталкивать клиента к быстрому оформлению заявки.

Результат от разработки:
- полный дизайн сайта
- адаптивная верстка
- рабочий калькулятор стоимости
- форма вызова эвакуатора
- готовность к подключению backend/API
